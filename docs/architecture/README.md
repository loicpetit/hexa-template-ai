# Architecture Documentation

This folder contains architecture documentation maintained by the Software Architect agent.

## Required baseline file
- architecture-overview.md

## Suggested additional files
- <domain>-architecture.md
- integration-architecture.md
- deployment-architecture.md

## Rules
- Keep this documentation aligned with approved technical requirements in technical-requirements/.
- Update architecture docs whenever architecture-impacting TREQs are created, replaced, or approved.
- Include links to architecture docs in TREQ metadata where relevant.
- Keep documentation user friendly: explain purpose and impact before low-level details.
- Use graphics when possible to improve understanding (prefer Mermaid diagrams for versioned docs).
